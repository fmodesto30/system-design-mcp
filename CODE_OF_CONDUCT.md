# Code of Conduct

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/) (v2.1) como
código de conduta.

Em resumo: trate todo mundo com respeito; sem assédio, ataque pessoal ou linguagem
discriminatória; foque em crítica técnica construtiva. Mantenedores podem remover/editar
contribuições que violem isso.

Comportamento inaceitável pode ser reportado abrindo uma issue ou contatando o mantenedor do repo.
O texto completo do Contributor Covenant está em
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
